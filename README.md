# IT Support Ticket Analysis Tool

An IT/security project that parses help desk tickets, auto-categorizes issues using keyword rules, and generates summary reports for triage and operations.

## Features
- Auto-categorizes tickets (network, VPN, password resets, phishing, etc.)
- Finds recurring issues and prioritizes high-risk tickets
- Exports a JSON report for operational review

## Tech
- Python
- CSV + JSON

## Run
```bash
python ticket_analyzer.py
